## Apache ShardingSphere概述

- Apache ShardingSphere 是一套开源的分布式数据库中间件解决方案组成的生态圈，
- 它由 JDBC（重点）、Proxy 和 Sidecar（规划中）这 3 款相互独立，却又能够混合部署配合使用的产品组成。
- 它们均提供标准化的数据分片、分布式事务和数据库治理功能，可适用于如 Java 同构、异构语言、云原生等各种多样化的应用场景
