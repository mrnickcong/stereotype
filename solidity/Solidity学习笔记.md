1. #### 状态变量：

   1. 变量声明的一般形式

      ```
      类型 [可见性] 变量名;
      举例：uint public x;
      ```

   2. 变量的可见性

      | 可见性关键字 | 可见范围                                      | 备注                                             |
      | ------------ | --------------------------------------------- | ------------------------------------------------ |
      | public       | 完全可见                                      | 虽然可见，但是只读不能写。需要写，再加方法实现。 |
      | private      | 对本合约可见、其他不可见                      |                                                  |
      | internal     | 对本合约及子合约可见，相当于Java中的protected | 变量默认可见性：internal                         |

      当变量的可见性为public时，编译器编译是会为其自动生成get的同名方法。

      合约外部：其他合约、链下的钱包地址
